# Soroban Escrow Payment DApp

## Project Description

Soroban Escrow Payment DApp is a decentralized fintech application built on Stellar Soroban smart contracts.

The project allows users to securely lock funds during online transactions. Funds are only released when both parties confirm the transaction conditions.

This helps reduce scams and increases trust in peer-to-peer payments.

---

## Project Vision

Our vision is to build a secure, low-cost, and transparent payment escrow platform for freelancers, online marketplaces, and digital services using Stellar blockchain technology.

The project aims to demonstrate how Soroban smart contracts can be applied in real-world fintech applications.

---

## Key Features

- Secure escrow payment using Soroban smart contracts
- Token transfer between users
- Lock and release payment mechanism
- Refund support for failed transactions
- Stellar testnet integration
- Smart contract deployment with Soroban Studio

---

## Technologies Used

- Rust
- Soroban SDK
- Stellar Testnet
- Soroban Studio

---

## Smart Contract Functions

### transfer_usdc()

Transfer token between users.

### deposit()

Lock funds into escrow.

### release_payment()

Release locked funds to recipient.

### refund()

Refund locked funds back to sender.
